# WasteAI – AI Driven Circular Waste Intelligence System

WasteAI is a smart city platform that uses AI computer vision and real-time analytics to improve waste segregation and recycling.

## Features
- AI waste classification using YOLOv8
- Real-time analytics using Pathway
- Citizen reward system and leaderboard
- Environmental impact tracking
- Smart city waste dashboard

## Tech Stack
YOLOv8 • PyTorch • ONNX Runtime • Pathway • React • Node.js • Express
